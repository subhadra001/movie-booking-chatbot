# Movie Booking Chatbot

A conversational movie ticket booking web application that allows users to search movies, view showtimes, reserve seats, and generate digital tickets through a chat-based interface.

## Features
- Natural language movie search
- Theater and showtime selection
- Interactive seat booking
- Simulated payment workflow
- QR-code ticket confirmation

## Tech Stack
- Node.js
- Express.js
- JavaScript / TypeScript
- HTML / CSS
- REST APIs

## How to Run Locally

1. Install dependencies
   npm install

2. Start the server
   npm run dev

3. Open in browser
   http://localhost:5000
